# 实例化
- python 内部实例化一个新的对象时，它其实都使用了自身的type 属性，并创造了一个新的实例对象；
- 定义一个类时，会默认继承 object；
- type 是实例化维度；object 是继承时间的产物；
- __new__ 用于创建实例对象； __call__ 用于创建类对象；
